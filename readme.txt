Programa testado no Matlab R2018b.
Toolbox necessarias:
Image Processing Toolbox - v10.3
Computer Vision System Toolbox - v8.2
Statistics and Machine Learning Toolbox - v11.4

A estrutura do diretorio deve possuir um arquivo main, 8 funcoes e uma pasta signatures.
Existem 40 pessoas com 24 assinaturas cada, que foram o limite de imagens que o moodle permitiu enviar.

Como usar:
Abra o arquivo main.m no matlab e precione run.


nota: Alterar o nome das funcoes ou das assinaturas ira causar falhas.


Dataset: https://cedar.buffalo.edu/NIJ/publications.html
